목표 : 매수,매도 시뮬레이션, 결과 저장, 결과 평가

1. 시뮬레이션 시작 버튼을 누르면 시뮬레이션이 시작된다. 시뮬레이션 시작 시 시드 머니를 입력한다.
2. 5분 마다 eth를 대상으로 "롱/숏 문의" 버튼을 통해 롱/숏을 결정하고 포지션 진입 및 기록, 포지션 진입할 때는 롱/숏의 근거도 기록한다.
3. 이미 롱/숏 포지션이 있는 경우 익절,손절이 올 때까지 기다린다.
4. 익절,손절이 오면 기록한다.
5. 기록은 trading/YYYYMMDD.md 파일에 저장한다.
6. 시뮬레이션이 끝나면 시뮬레이션 결과를 trading/YYYYMMDD.md 파일에 저장한다.
